# v0.2.0 — 项目模板

## 目标

在 `templates/` 目录下创建两个独立的项目模板，每个模板有自己完整的 tsconfig（不继承根目录），可直接作为 workspace 包运行。

  - dependencies: @nestjs/common、@nestjs/core、@nestjs/platform-express、reflect-metadata、rxjs

  - scripts: dev（nest start --watch）、build（nest build）、start（node dist/main）、typecheck（tsc -p tsconfig.build.json --noEmit）

- [ ] `src/main.ts` — 应用入口，监听端口（默认 3001），设置全局前缀 `/api`，启用 CORS

- [ ] `src/app.controller.ts` — 根控制器，`GET /health` 端点
