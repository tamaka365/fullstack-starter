# Tooltip

悬停提示组件，基于 `Popover` 封装。children 包裹 API，白色背景样式内置。

## Props

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `content` | `ReactNode` | — | 提示内容 |
| `children` | `ReactElement` | — | 触发元素，需可接受 ref |
| `placement` | `Placement` | `'top'` | 弹出位置 |
| `openDelay` | `number` | `100` | 显示延迟（ms） |
| `closeDelay` | `number` | `0` | 隐藏延迟（ms） |
| `disabled` | `boolean` | `false` | 禁用时直接渲染 children |
