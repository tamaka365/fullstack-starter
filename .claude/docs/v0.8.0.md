# v0.8.0 — Popover 组件

## 目标

在 `packages/ui` 中实现 `Popover` 通用浮层原语。负责浮层定位、触发逻辑和动画，
不内置任何业务内容，调用方通过 `content` prop 自由注入。
后续 SidebarNav 折叠态的 hover 子菜单将直接复用本组件。

---

## API

```tsx
<Popover content={<SomePanel />} placement="right-start">
  <button>触发元素</button>
</Popover>
```

### PopoverProps

| prop            | 类型                  | 默认值          | 说明                                          |
| --------------- | --------------------- | --------------- | --------------------------------------------- |
| `children`      | `React.ReactElement`  | —               | 触发元素（anchor），必须能接收 ref 和事件     |
| `content`       | `React.ReactNode`     | —               | 浮层内容，调用方完全控制                      |
| `placement`     | `Placement`           | `'right-start'` | floating-ui 标准方位，支持全部 12 个方向      |
| `offset`        | `number`              | `8`             | 浮层与 anchor 的间距（px）                    |
| `open`          | `boolean`             | —               | 受控 open 状态；不传则内部维护                |
| `onOpenChange`  | `(open: boolean) => void` | —           | 受控时的状态变更回调                          |
| `openDelay`     | `number`              | `100`           | hover 显示延迟（ms），防止鼠标掠过误触        |
| `closeDelay`    | `number`              | `150`           | hover 隐藏延迟（ms），给用户从 anchor 移到浮层的时间 |

### 类型来源

`Placement` 直接从 `@floating-ui/react` 重新导出，调用方不需要单独安装。

## 行为规格

### Hover 安全区

使用 `safePolygon()` 作为 `useHover` 的 `handleClose`：
在 anchor 和浮层之间绘制三角形安全区，鼠标在区域内移动时不关闭浮层。
这是 "子菜单不闪烁" 的关键。

### 动画

- 浮层使用 `useTransitionStatus` 获取 `isMounted` 和 `status`
- 仅在 `isMounted` 时渲染 DOM（不挂载时完全从 DOM 中移除）
- `status` 取值：`'initial' | 'open' | 'close'`
- CSS 根据 `data-status` attribute 控制 opacity + scale 过渡：

```
initial → opacity 0, scale 0.95
open    → opacity 1, scale 1   （0.15s ease）
close   → opacity 0, scale 0.95（0.1s ease）
```

### Portal

使用 `FloatingPortal`（floating-ui 提供）渲染到 `document.body`，
避免父级 `overflow: hidden` 或 `z-index` 栈裁剪浮层。

### flip / shift

使用 floating-ui 的 `flip()` 和 `shift()` middleware：
- `flip`：浮层超出视口时翻转到对侧
- `shift`：翻转后仍超出时沿轴平移，保证浮层始终可见

### 步骤 3 — Popover.css.ts

```ts
// panel 基础样式（位置由 floating-ui 内联 style 控制）
export const panel = style({
  zIndex: 1000,
  transition: 'opacity 0.15s ease, transform 0.15s ease',
  transformOrigin: 'var(--transform-origin, left center)',
})

// status = 'open'
export const panelOpen = style({
  opacity: 1,
  transform: 'scale(1)',
})

// status = 'initial' | 'close'
export const panelClose = style({
  opacity: 0,
  transform: 'scale(0.95)',
  // close 状态缩短动画时长
  transitionDuration: '0.1s',
})
```

### 步骤 4 — Popover.tsx

关键实现骨架：

```tsx
export function Popover({
  children,
  content,
  placement = 'right-start',
  offset: offsetValue = 8,
  open: controlledOpen,
  onOpenChange,
  openDelay = 100,
  closeDelay = 150,
}: PopoverProps) {
  const isControlled = controlledOpen !== undefined
  const [internalOpen, setInternalOpen] = useState(false)
  const open = isControlled ? controlledOpen : internalOpen

  const handleOpenChange = useCallback(
    (next: boolean) => {
      if (!isControlled) setInternalOpen(next)
      onOpenChange?.(next)
    },
    [isControlled, onOpenChange],
  )

  const { refs, floatingStyles, context } = useFloating({
    open,
    onOpenChange: handleOpenChange,
    placement,
    middleware: [offsetMiddleware(offsetValue), flip(), shift()],
    whileElementsMounted: autoUpdate,
  })

  const hover = useHover(context, {
    delay: { open: openDelay, close: closeDelay },
    handleClose: safePolygon(),
  })
  const { getReferenceProps, getFloatingProps } = useInteractions([hover])

  const { isMounted, status } = useTransitionStatus(context, { duration: 150 })

  return (
    <>
      {cloneElement(children, {
        ref: refs.setReference,
        ...getReferenceProps(),
      })}
      {isMounted && (
        <FloatingPortal>
          <div
            ref={refs.setFloating}
            style={floatingStyles}
            className={clsx(panel, status === 'open' ? panelOpen : panelClose)}
            {...getFloatingProps()}
          >
            {content}
          </div>
        </FloatingPortal>
      )}
    </>
  )
}
```

## 注意事项

- `children` 必须是单个 React 元素（不能是字符串或数组），`cloneElement` 需要注入 ref
- 若 `children` 是函数组件且未 `forwardRef`，floating-ui 会警告 → 文档中提示调用方用原生元素或已 forwardRef 的组件
- `autoUpdate` 负责滚动/resize 时重新计算位置，窗口滚动时浮层不会错位
- vanilla-extract 不支持 CSS 变量作为 `transformOrigin` 的动态值，改用内联 style 或固定值
