# v0.6.0 — Layout 组件

## 目标

在 `packages/ui` 中实现 `Layout` React 组件，通过 props 组合覆盖所有经典布局场景，使用 vanilla-extract 管理样式。

---

## 文件结构

```
packages/ui/src/
  layout/
    Layout.css.ts     # vanilla-extract 样式
    Layout.tsx        # 组件实现
  index.ts            # 新增 Layout 导出
```

---

```bash
cd packages/ui
```

```json
"peerDependencies": {
```

## Step 2 — 编写 Layout.css.ts

### 全局初始化

```ts
globalStyle('html, body', {
  margin: 0,
  padding: 0,
})
```

随 Layout CSS 引入时生效，非独立全局 reset。

### 样式类

| 类名 | 作用 |
|------|------|
| `root` | `display: flex; min-height: 100vh` |
| `row` | `flex-direction: row` |
| `column` | `flex-direction: column` |
| `body` | 内层容器，`display: flex; flex: 1` |
| `header` | `flex-shrink: 0` |
| `footer` | `flex-shrink: 0` |
| `sidebar` | `flex-shrink: 0` |
| `content` | `flex: 1` |
| `stickyHeader` | `position: sticky; top: 0; z-index: 10` |
| `stickySidebar` | `position: sticky; top: 0; max-height: 100vh; overflow-y: auto` |

## Step 3 — 编写 Layout.tsx

### Props

```ts
interface LayoutProps {
  sidebarFull?: boolean       // sidebar 撑满整体高度（在 header/footer 外侧），默认 false
  header?: React.ReactNode
  footer?: React.ReactNode
  sidebar?: React.ReactNode
  stickyHeader?: boolean      // 默认 true
  stickySidebar?: boolean     // 默认 true
  children?: React.ReactNode
}
```

> `LayoutMode` 类型不再需要，全部由 props 组合决定结构。

### 布局逻辑

**`sidebarFull: false`（默认）— sidebar 嵌套在 header 下方：**

```
<div row|column>        ← 有 sidebar 时 row，否则 column
  <header />            ← 仅在 column 根时存在（无 sidebar 或 sidebarFull）
  <div row flex:1>      ← 有 sidebar 时的 body 层
    <aside />
    <main />
  </div>
  <footer />
</div>
```

实际结构：

```
column root:
  header
  row body (flex:1):
    sidebar
    main (flex:1)
  footer
```

**`sidebarFull: true` — sidebar 撑满整体高度（在 header/footer 外侧）：**

```
row root:
  sidebar
  column body (flex:1):
    header
    main (flex:1)
    footer
```

**无 sidebar 时**（`sidebarFull` 无意义）：

```
column root:
  header
  main (flex:1)
  footer
```
