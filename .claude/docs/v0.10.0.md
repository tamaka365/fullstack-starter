# v0.10.0 — Tooltip 组件

## 概述

在 `packages/ui` 中实现 `Tooltip` 组件。基于 `Popover` 封装，hover 触发，children 包裹 API，白色背景样式内置，供 SidebarNav 折叠态及其他场景复用。

## API 设计

```tsx
<Tooltip content="菜单" placement="right">
  <button>hover me</button>
</Tooltip>
```

### TooltipProps

| prop        | 类型              | 默认值  | 说明                                      |
| ----------- | ----------------- | ------- | ----------------------------------------- |
| `content`   | `React.ReactNode` | —       | tooltip 显示内容                          |
| `children`  | `React.ReactElement` | —    | 触发元素，必须是单个可接受 ref 的元素     |
| `placement` | `Placement`       | `'top'` | 浮层方位，同 Popover                      |
| `openDelay` | `number`          | `100`   | 显示延迟（ms），同 Popover 默认           |
| `closeDelay`| `number`          | `0`     | 隐藏延迟（ms）                            |
| `disabled`  | `boolean`         | `false` | 禁用时直接渲染 children，不挂载 Popover   |

### children 约束

`React.cloneElement(children, props)` 将 floating-ui 的 `ref` 和事件处理器注入到子元素。
- 原生 DOM 元素（`button`、`a`、`div` 等）直接支持
- 自定义组件需 `React.forwardRef` 转发 ref

## 实现步骤

### 步骤 1：types.ts

新建 `packages/ui/src/tooltip/types.ts`，定义 `TooltipProps`。

### 步骤 2：Tooltip.css.ts

新建 `packages/ui/src/tooltip/Tooltip.css.ts`，定义 `tooltipContent` class：
- `background: '#fff'`
- `borderRadius: 6`
- `boxShadow: '0 2px 8px rgba(0,0,0,0.15)'`
- `padding: '6px 10px'`
- `fontSize: '0.875rem'`
- `whiteSpace: 'nowrap'`

### 步骤 3：Tooltip.tsx

新建 `packages/ui/src/tooltip/Tooltip.tsx`：
- `'use client'`
- `disabled` 时直接返回 `children`
- 内部使用 `Popover`，`triggers="hover"`，`placement` / `openDelay` / `closeDelay` 透传
- `trigger` render prop 用 `React.cloneElement(children, props)` 注入 ref 和事件
- `content` 包裹在 `<div className={styles.tooltipContent}>` 中

### 步骤 4：导出

在 `packages/ui/src/index.ts` 添加：
```ts
export { Tooltip } from './tooltip/Tooltip'
export type { TooltipProps } from './tooltip/types'
```
