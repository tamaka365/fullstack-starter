# v0.12.0 — SidebarNav 折叠态级联弹出菜单

## 概述

新建 `CascadeMenu` 组件，用于 SidebarNav 折叠态弹出菜单。菜单项有子节点时，hover 触发下一级 Popover（递归级联），不使用折叠展开。无子节点时渲染为普通链接/按钮。

## 组件设计

### API

```tsx
<CascadeMenu
  items={MenuItemData[]}
  activeKey={string}
  renderAs={React.ElementType}
  getItemProps={(item) => Record<string, unknown>}
/>
```

复用 `MenuItemData` 类型，与 `MenuList` 保持一致，`SidebarNav` 无需额外数据转换。

### 渲染逻辑

- **叶子节点**（`!hasChildren`）：用 `renderAs` 渲染为链接/按钮，spread `getItemProps(item)`
- **父节点**（`hasChildren`）：`Popover triggers="hover" placement="right-start"`，trigger 为一个 div 行（含 label + ChevronRight），content 为递归嵌套的 `<CascadeMenu items={item.children} />`

### 自带背景包装

`CascadeMenu` 根元素自带白色背景容器（同 `popoverMenu`），让每一层弹出都能正确显示背景。

## 涉及文件

- `packages/ui/src/cascade-menu/CascadeMenu.css.ts`：`menu`（白色背景容器）、`list`（ul reset）、`item`（行样式）、`itemActive`、`itemDisabled`、`icon`、`chevron`（ChevronRight wrapper）
- `packages/ui/src/cascade-menu/CascadeMenu.tsx`：`CascadeMenu` 组件，递归渲染
- `packages/ui/src/index.ts`：导出 `CascadeMenu` 及 `CascadeMenuProps`
- `packages/ui/src/sidebar/SidebarNav.tsx`：折叠态弹出 content 从 `<div popoverMenu><MenuList /></div>` 改为 `<CascadeMenu />`；移除 `popoverMenu` 引用
- `packages/ui/src/sidebar/SidebarNav.css.ts`：删除 `popoverMenu` class（移入 CascadeMenu）
